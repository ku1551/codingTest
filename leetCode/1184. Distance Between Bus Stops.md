# 1184. Distance Between Bus Stops

## Source

[leetcode 1184](https://leetcode.com/problems/distance-between-bus-stops/description/)

## Apporach
- 원형을 이루는 정거장들이 있고 시작점에서 목표지점까지 최소한의 거리를 구한다. 
주어진 변수 거리배열(int[]) , 시작지점(int), 도착지점(int) 


1. 원형의 정거장임을 기억한다. 
2. 원형의 특성으로 방향과 관계 없이 짧은거리와 긴거리의 합은 모든 거리의 합과 같다. 
3. 시작점과 도착점의 위치가 바뀌어도 거리와는 관계가 없다. 

-  실 작업
1. 시작점과 도착점 중 낮은 지점을 시작점으로 높은 수를 도착점으로 재설정한다. 
2. 전체 원의 길이를 구한다. 
3. 시작점에서 도착점까지의 길이를 구한다. 
4. 전체 거리를 구한다. 
5. 전제 거리에서 시작점에서 도착점까지의 길이를 뺀다. 
6. 두 값중 작은 값을 반환한다.


## Code

    class Solution {
        public int distanceBetweenBusStops(int[] distance, int start, int destination) {
            int all = Arrays.stream(distance).sum();

            if(start > destination){
                int tmp = start;
                start = destination;
                destination = tmp;
            }
            
            int len = 0;

            for(int i=start; i<destination; i++){
                len += distance[i];
            }

            return all - len > len ? len : all - len;
        }
    }

## 최적화

    class Solution {
        public int distanceBetweenBusStops(int[] distance, int start, int destination) {
            int all = Arrays.stream(distance).sum();

            if(start > destination){

                start = destination;
                destination = tmp;

            }

            return Math.min(all-len,len);
        }
