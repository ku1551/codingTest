# 3498. Reverse Degree of a String

## Source

[leetcode 3498](https://leetcode.com/problems/reverse-degree-of-a-string/description/)

## Apporach
- 알파벳의 위치를 구한뒤 27에서 값을 빼준다. * 인덱스 +1 값과 곱한뒤 더하면 결과값 

## Code

    class Solution {
        public int reverseDegree(String s) {
            
            int result = 0;

            for(int i=0; i<s.length(); i++){
                
                int num = s.charAt(i) -'a' + 1;

                result += (i+1) * (27- num);
            }
            return result;
        }
    }

## 최적화

    class Solution {
        public int reverseDegree(String s) {
            long result = 0;
            for (int i = 0; i < s.length(); i++) {
                int rev = ('z' - s.charAt(i)) + 1;
                result += (long)(i + 1) * rev;
            }
            return (int) result;
        }
    }
