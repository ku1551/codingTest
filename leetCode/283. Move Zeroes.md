# 283. Move Zeroes

## Code
    class Solution {
        public void moveZeroes(int[] nums) {
            int left = 0;

            for(int i = 0; i<nums.length; i++){
                if( nums[i] != 0){
                    int tmp = nums[left];
                    nums[left] = nums[i];
                    nums[i] = tmp;
                    left++;
                }
            }
        }
    }
